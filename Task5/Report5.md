# Домашнее задание №2. «Автоматическая очистка и предобработка данных»
*студент: Корецкий В.П.*

## Задание состоит из двух частей:
* Автоматическое выявление проблем с данными с помощью ИИ.
* Очистка данных: удаление дубликатов, заполнение пропусков и исправление выбросов.
  
## Цель задания
Цель задания — научиться применять ИИ-инструменты для автоматического выявления и устранения проблем в данных (пропуски, дубликаты, аномалии), а также оценивать качество очистки.

## Условия задания
Вам предоставлен датасет «Датасет_для_задания_автоматическая_очистка.csv» (100 строк) с данными клиентов онлайн-магазина.

## Структура данных:
* ID — уникальный идентификатор
* Age — возраст (в годах)
* Income — доход (в тыс. руб.)
* Purchase — сумма покупки (в руб.)
* City — город проживания
* Gender — пол
* Last_Purchase_Days — дней с последней покупки
* Category — категория покупки

## Особенности датасета:
* Пропуски в Age, Income, Purchase
* Полные дубликаты записей (строки 1–30 повторяются как 31–60 и 61–90)
* Выбросы:
1. Age = 150
2. Income = 20
3. Purchase = 500000
4. Last_Purchase_Days = 999
  
## Задача 1. Автоматическое выявление проблем с помощью ИИ
**Что нужно сделать:**
1. Напишите промпт так, чтобы:
* Определить количество пропусков по каждому столбцу
* Найти дубликаты (полные и частичные)
* Выявить аномалии в числовых полях (Age, Income, Purchase, Last_Purchase_Days)
* Получить рекомендации по очистке данных

*Пример запроса:*  
«Проанализируй датасет на наличие пропусков, дубликатов и аномалий. Укажи:
* Сколько пропусков в каждом столбце?
* Есть ли полные или частичные дубликаты?
* Какие значения в полях Age, Income, Purchase, Last_Purchase_Days являются выбросами?
* Дай рекомендации по очистке данных для каждого типа проблемы.»
  
Для более стабильных результатов делайте решение одной задачи одним промптом.  
Запишите ответ ИИ:
* Сколько пропусков обнаружено?
* Сколько дубликатов найдено?
* Какие значения ИИ определил как выбросы?

## Задача 2. Очистка данных пошагово
### Шаг 1: Удаление дубликатов
Что нужно сделать:  
Данные содержат повторяющиеся записи. Нужно удалить дубликаты, чтобы каждая запись была уникальной.
  
*Промпт для ИИ:*
«Найди и удали полные дубликаты строк в датасете, кроме первого вхождения.»

### Шаг 2: Заполнение пропусков
Что нужно сделать:  
Заполните пустые значения, чтобы данные стали полным:
* Для числовых столбцов (Age, Income, Purchase) используйте медиану или среднее
* Для категориальных (City) — самое частое значение
  
*Промпт для ИИ:*  
«Как заполнить пропуски в столбцах Age, Income и Purchase? Используй простой метод (без кода). Объясни, почему этот метод подходит.»

## Шаг 3: Исправление выбросов
Что нужно сделать:  
В данных есть нереалистичные значения (например, возраст 150 лет). Нужно их исправить.

Промпт для ИИ:
«Найди выбросы в числовых полях (Age, Income, Purchase). Предложи корректировку:
* Заменить слишком маленькие/большие значения на границы допустимого диапазона
* Или удалить строки с экстремальными значениями.»


## Как сдать задание на проверку
Чтобы получить зачёт, представьте решение в структурированном и понятном виде.

1. Формат ответа

4. Оформление
